# TalkQuest — Docs Vault

This folder is a shared **Obsidian-style vault** for the TalkQuest project. It's just plain
`.md` files, so you edit them in Obsidian and Claude edits them here — same files, no sync
tooling needed. Open this `docs/` folder as an Obsidian vault to browse with `[[links]]`.

## Map of Content

- [[functional-spec]] — v1 / MVP functional spec (core concept, flows, data model, anti-cheat, AI pipeline)
- [[board]] — Trello-like Kanban board: Backlog / To Do / In Progress / Done
- [[decisions]] — running log of key product & technical decisions
- [[progress]] — what's done / in progress / next

## Conventions

- Link notes with `[[note-name]]` (no `.md`, no folder path).
- Use Mermaid code blocks for diagrams — renders in both Obsidian and here.
- Keep one topic per note; split when a note gets too big.
